# Algoritmo-Nearest-Neighbor

```markdown
# Estudos em C

Este é um código capaz de calcular uma rota eficiente (não a melhor) de entregas em um ou mais pontos.
Atráves de coordenadas X e Y inseridas pelo usúario (pode ser executado juntamente com o arquivo entradas1.txt),
é calculado a entrega mais próxima não visitada, no o programa fim volta ao ponto de origem. 

## O que eu vou achar?

- Estruturas de repetição
- Funções
- Vetores e matrizes
- Ponteiros

## Como compilar

Utilize o CMD e vá até a pasta onde está localizada o programa e digite:
gcc main.c -o main
main.exe

Utilizando o arquivos entradas1.txt fica:
gcc main.c -o main
main.exe < entradas1.txt

O resultado esperado (caso utilize o entradas1.txt) é que o "Percurso Total" seja 43.69km

Faça bom proveito!
